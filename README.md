# 验证码识别

## 安装
测试环境：windows+python2.7.12

直接依赖：

+ PIL
+ sklearn

环境配置步骤：

1. 安装 [Anaconda](https://www.continuum.io/downloads)
2. 安装PIL，命令行下pip install pillow
3. 安装sklearn，命令行下pip install scikit-learn


## 使用
进入src目录，运行命令`python *.py`
